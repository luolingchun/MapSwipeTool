# MapSwipeTool

QGIS 插件——地图卷帘工具，参考已有的[地图卷帘工具](https://github.com/lmotta/mapswipetool_plugin)，做了一些优化：

- 图层组支持
- 图层选择
- 操作优化
- 图层隐藏

## 安装说明

- [下载 zip 包（QGIS 3.x）](https://codeload.github.com/luolingchun/MapSwipeTool/zip/refs/tags/v1.2.3)
- [下载 zip 包（QGIS 4.x）](https://github.com/luolingchun/MapSwipeTool/archive/refs/heads/master.zip)

使用 QGIS 中的 Install from ZIP 安装

## 使用说明

- 点击鼠标左键并拖动：开始卷帘
- 按下`Ctrl`键同时点击鼠标左键：隐藏上方图层
